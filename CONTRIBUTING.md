# Contributing to this Project

Firstly, thank you for your interest in contributing to the AntRig project!
It is the goal of this project to provide the research community with a powerful, flexible, yet cost-effective measurement system solution.
By collecting all the ideas and contributions from different teams within the open-source approach, we hope to develop a system that as versatile and useful as possible.
It is your contributions that will help to achieve this goal!

## How to Contribute
1. **Fork** this repository to your own GitHub account.  
2. Create a **new branch** for your changes (`git checkout -b feature-your-idea`).  
3. Make your changes and commit them with clear commit messages.  
4. **Push** your branch to your fork.  
5. Open a **Pull Request** against the `main` branch of this repository.  

Before creating a Pull Request, please make sure your code is clean and introduces no errors.
In regard to code style, please use doc-strings, self-explaining variable/method names and write meaningful commit messages to simplify the review process.

_Disclaimer:_\
For now, the 'Technical Committee' of this project only consists of the project owner, so please be patient with the review process.
It might take some time until your Pull Request is reviewed and merged, but be assured that it will be done as soon as possible.
We are happy about every contribution and discussion that comes up!

### Discussions & Questions
If you are unsure or want to discuss an idea, please open an **Issue** in this repository.
It's the easiest way to exchange new ideas and requirements for the system and finally this kind of exchange guides 
the development of the project in the most useful direction.
